# About FoundByMe

**FoundByMe**는 개인의 로컬 학습 자료를 효율적으로 관리하고 검색할 수 있도록 돕는 **오픈소스 시맨틱 검색 엔진**입니다.

단순한 키워드 매칭을 넘어, 문맥을 이해하는 **RAG(Retrieval-Augmented Generation)** 기술을 통해 강의 자료, PDF, 코드 조각 등 흩어진 지식을 자연어로 쉽게 찾을 수 있습니다.

---

## 🎨 Custom Theme Development

이 웹사이트는 FoundByMe 프로젝트의 정체성을 나타내기 위해 **직접 제작 및 재설계한 커스텀 Jekyll 테마**를 사용하고 있습니다. 기존 테마를 단순히 적용하는 것에 그치지 않고, 구조와 디자인을 전면적으로 개편하여 독창성과 사용성을 높였습니다.

### 🛠️ Key Innovations (주요 혁신 사항)

**1. 테마 독립화 및 구조 재설계**
기존의 `remote_theme` 의존성을 제거하고, 소스 코드를 로컬 환경으로 이관하여 완전히 독립적인 테마 구조를 구축했습니다. 이를 통해 자유로운 기능 확장과 유지보수가 가능해졌습니다.

**2. 브랜드 아이덴티티 확립 (Design System)**
* **Color:** 세종대학교의 상징색인 **Crimson Red (#c3002f)**와 **Slate Grey (#51626f)**를 메인 컬러로 정의하여 강렬하고 전문적인 인상을 주었습니다.
* **Typography:** 가독성이 뛰어난 구글 폰트 **'Inter'**를 적용하여 기술 문서에 최적화된 타이포그래피를 구현했습니다.

**3. UX/UI 구조 혁신**
* **Hero Banner:** 메인 페이지 상단에 프로젝트의 비전을 직관적으로 전달하는 대형 배너와 CTA(Call to Action) 버튼을 배치했습니다.
* **Card UI:** 밋밋한 텍스트 리스트 대신 Grid 레이아웃 기반의 **카드형 디자인**을 도입하여 정보 전달력을 강화했습니다.
* **Custom Footer:** 학교 정보와 팀원 연락처, 소셜 링크를 포함한 확장형 푸터를 직접 코딩하여 적용했습니다.

**4. 디테일 강화**
* **Search Experience:** 검색 중심 프로젝트답게 검색창 디자인을 고도화하고 포커스 효과를 추가했습니다.
* **User Friendly:** 커스텀 404 페이지와 파비콘(Favicon)을 적용하여 사용자 경험의 완성도를 높였습니다.
