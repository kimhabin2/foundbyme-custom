# Usage

This guide explains how to search and utilize local files via the FoundByMe Web Interface (Web UI).

---

## 1Ô∏è‚É£ Indexing

To perform a search, you must first register your computer's files to FoundByMe.

1. Go to the **Settings Page**.
2. Enter the local folder path you want to search in the **'Path'** input field. (e.g., `C:/Users/Name/Documents`)
3. Click the **'Start Indexing'** button.
4. Wait until the progress bar reaches 100%.

**Supported File Formats**
Supports PDF(`.pdf`), Text(`.txt`, `.md`), Source Code(`.py`, `.js`, `.java`), Images(`.png`, `.jpg`), and more.
{: .callout-note }

If there are too many search results, use filters.

## 3Ô∏è‚É£ Utilizing Results

The search result card displays the following information:

* **Summary:** AI summarizes what the document is about.
* **Preview:** Check key contents without opening the file.
* **Score:** Shows a score of how relevant it is to your question.

**Open Directly**
Clicking the result card immediately opens the file in the default viewer.
