# Feature Showcase

<div class="feature-grid">
  <div class="feature-card">
    <h3>🔍 Segment-Level Indexing</h3>
    <p>문서를 통째로 저장하지 않고 의미 단위(Chunk)로 분할하여 ChromaDB에 저장합니다. 파일명뿐만 아니라 페이지 번호까지 정확히 찾아냅니다.</p>
  </div>
  <div class="feature-card">
    <h3>🤖 Hybrid Backend</h3>
    <p>ChromaDB(벡터)와 PostgreSQL(메타데이터)을 결합하여 강력한 검색 성능과 데이터 무결성을 보장합니다.</p>
  </div>
  <div class="feature-card">
    <h3>📊 3D PCA Visualization</h3>
    <p>Scikit-learn의 PCA를 활용하여 고차원 임베딩 벡터를 3차원으로 축소, 검색 결과와 문서 간의 관계를 시각적으로 보여줍니다.</p>
  </div>
</div>
